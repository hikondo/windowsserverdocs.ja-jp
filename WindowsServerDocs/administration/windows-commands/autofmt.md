# <a name="autofmt"></a>autofmt

自動ファイルシステムフォーマットユーティリティを起動します。これにより、ドライブまたはパーティションが Windows 回復コンソールから呼び出されたときにフォーマットされます。

> [!IMPORTANT]
> コマンドラインから直接**autofmt**を実行することはできません。
