# <a name="bitsadmin-cache-and-getlimit"></a>bitsadmin cache および getlimit

キャッシュ制限を取得します。

## <a name="syntax"></a>構文

```
bitsadmin /cache /getlimit
```

## <a name="examples"></a>例

キャッシュ制限を取得するには:

```
bitsadmin /cache /getlimit
```
