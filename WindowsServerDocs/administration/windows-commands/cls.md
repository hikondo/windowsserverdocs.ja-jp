# <a name="cls"></a>cls



コマンドプロンプトウィンドウをクリアします。

このコマンドを使用する方法の例については、[例](#BKMK_examples)を参照してください。

## <a name="syntax"></a>構文

```
cls
```

## <a name="parameters"></a>パラメーター

|パラメーター|説明|
|---------|-----------|
|/?|コマンド プロンプトにヘルプを表示します。|

## <a name="BKMK_examples"></a>例

コマンドプロンプトウィンドウに表示されたすべての情報をクリアし、空白のウィンドウに戻るには、次のように入力します。
```
cls 
```
