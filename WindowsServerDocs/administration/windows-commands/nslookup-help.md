# <a name="nslookup-help"></a>nslookup help

サブコマンドのヘルプテキストを表示します。

## <a name="syntax"></a>構文

```
help
```

```
?
```

### <a name="parameters"></a>パラメーター

| パラメーター | 説明 |
| --------- | ----------- |
| /? | コマンド プロンプトにヘルプを表示します。 |
| /help | コマンド プロンプトにヘルプを表示します。 |
