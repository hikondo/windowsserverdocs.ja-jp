# <a name="bitsadmin-peers-and-clear"></a>bitsadmin ピアとクリア



ピアの一覧をクリアします。

## <a name="syntax"></a>構文

```
bitsadmin /Peers /Clear 
```

## <a name="BKMK_examples"></a>例

次の例では、ピアリストをクリアします。
```
C:\>bitsadmin /Peers /Clear
```
