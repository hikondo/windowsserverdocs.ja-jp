# <a name="bitsadmin-peers-and-clear"></a>bitsadmin peers および clear

ピアの一覧をクリアします。

## <a name="syntax"></a>構文

```
bitsadmin /peers /clear
```

## <a name="examples"></a>例

ピアリストをクリアします。

```
bitsadmin /peers /clear
```
