# <a name="fveupdate"></a>fveupdate

FveUpdate は、コンピューターのアップグレード時にセットアッププログラムによって使用される内部ツールです。 最新バージョンには、BitLocker に関連付けられているメタデータを更新します。 このツールとは独立して実行することはできません。
