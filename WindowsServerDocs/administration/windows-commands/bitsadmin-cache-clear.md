# <a name="bitsadmin-cache-and-clear"></a>bitsadmin cache および clear

ローカルキャッシュを削除します。

## <a name="syntax"></a>構文

```
bitsadmin /cache /clear
```

## <a name="examples"></a>例

ローカルキャッシュを削除するには:

```
bitsadmin /cache /clear
```
