# <a name="bitsadmin-peers-and-discover"></a>bitsadmin peers および discover

ピアを再び検出します。

## <a name="syntax"></a>構文

```
bitsadmin /peers /discover
```

## <a name="examples"></a>例

ピアを再び検出するには:

```
bitsadmin /peers /discover
```
