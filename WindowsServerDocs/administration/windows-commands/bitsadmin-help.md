# <a name="bitsadmin-help"></a>bitsadmin help



コマンドラインの使用法を表示します。

## <a name="syntax"></a>構文

```
bitsadmin /help | /?
```

## <a name="BKMK_examples"></a>例

次の例では、コマンドラインヘルプを取得します。
```
C:\>bitsadmin /help
```
