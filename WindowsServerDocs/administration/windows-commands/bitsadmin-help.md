# <a name="bitsadmin-help"></a>bitsadmin help

Bitsadmin のコマンドラインパラメーターおよびオプションに関するヘルプ関連情報を表示します。

## <a name="syntax"></a>構文

```
bitsadmin /help | /?
```

## <a name="examples"></a>例

を使用すると、コマンドラインヘルプを取得できます。

```
bitsadmin /help
```
