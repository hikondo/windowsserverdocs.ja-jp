# <a name="bitsadmin-cache-and-list"></a>bitsadmin cache and list



すべてのキャッシュエントリを一覧表示します。

## <a name="syntax"></a>構文

```
bitsadmin /Cache /List [/Verbose] 
```

## <a name="BKMK_examples"></a>例

次の例では、すべてのキャッシュエントリを詳細な形式で一覧表示します。
```
C:\> bitsadmin /Cache /List /Verbose
```
