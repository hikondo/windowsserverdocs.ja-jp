# <a name="bitsadmin-cache-and-list"></a>bitsadmin cache および list

すべてのキャッシュエントリを一覧表示します。

## <a name="syntax"></a>構文

```
bitsadmin /cache /list [/verbose]
```

## <a name="examples"></a>例

を指定すると、すべてのキャッシュエントリが詳細な形式で一覧表示されます。

```
bitsadmin /cache /list /verbose
```
