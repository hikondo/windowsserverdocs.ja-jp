# <a name="bitsadmin-peers-and-list"></a>bitsadmin peers および list

すべてのピアを一覧表示します。

## <a name="syntax"></a>構文

```
bitsadmin /peers /list
```

## <a name="examples"></a>例

すべてのピアを一覧表示するには:

```
bitsadmin /peers /list
```
