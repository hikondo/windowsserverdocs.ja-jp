# <a name="nslookup-set-all"></a>nslookup set all

既定のサーバーとコンピューター (ホスト) を含む、現在の構成設定値を出力します。

## <a name="syntax"></a>構文

```
set all
```

### <a name="parameters"></a>パラメーター

| パラメーター | 説明 |
| --------- | ----------- |
| /? | コマンド プロンプトにヘルプを表示します。 |
| /help | コマンド プロンプトにヘルプを表示します。 |
