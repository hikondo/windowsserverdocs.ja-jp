# <a name="nslookup-exit"></a>nslookup/exit

Nslookup コマンドラインツールを終了します。

## <a name="syntax"></a>構文

```
nslookup /exit
```

### <a name="parameters"></a>パラメーター

| パラメーター | 説明 |
| --------- | ----------- |
| /? | コマンド プロンプトにヘルプを表示します。 |
| /help | コマンド プロンプトにヘルプを表示します。 |
