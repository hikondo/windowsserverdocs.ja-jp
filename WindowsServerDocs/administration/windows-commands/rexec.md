# <a name="rexec"></a>rexec

指定されたコマンドをリモートホスト上で実行します。 リモートホストは、rexec が接続するために rexecd サービス (またはデーモン) を実行している必要があります。

> [!IMPORTANT]
> **Rexec**コマンドは非推奨とされており、Windows でサポートされるとは限りません。
