# <a name="ftp-status"></a>ftp status

> 適用対象: Windows Server (半期チャネル)、Windows Server 2019、Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

Ftp 接続の現在の状態を表示します。

## <a name="syntax"></a>構文

```
status
```

### <a name="examples"></a>例

Ftp 接続の現在の状態を表示するには、次のように入力します。

```
status
```
