# <a name="automount"></a>オートマ



自動マウント機能を有効または無効にします。 有効にすると (既定)、システムに新しいベーシックボリュームまたはダイナミックボリュームが追加されると、Windows によって自動的にファイルシステムがマウントされ、ボリュームにドライブ文字が割り当てられます。
