# <a name="bitsadmin-cache-and-help"></a>bitsadmin cache および help

**キャッシュ**スイッチのコマンドラインの使用法を表示します。

## <a name="syntax"></a>構文

```
bitsadmin /cache /help
```

## <a name="examples"></a>例

を使用すると、 **キャッシュ** スイッチのコマンドラインヘルプを表示できます。

```
bitsadmin /cache /help
```
