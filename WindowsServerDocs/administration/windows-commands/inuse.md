# <a name="inuse"></a>inuse

Inuse コマンドは非推奨とされており、Windows の将来のリリースでサポートされるとは限りません。

このツールは、Windows Server 2003 に含まれます。 詳細については、「 [Inuse](/previous-versions/orphan-topics/ws.10/dd996699(v=ws.10))」を参照してください。
