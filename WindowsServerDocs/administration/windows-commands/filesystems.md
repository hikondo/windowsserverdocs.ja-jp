# <a name="filesystems"></a>ファイルシステム



フォーカスがあるボリュームの現在のファイルシステムに関する情報を表示し、ボリュームのフォーマットがサポートされているファイルシステムの一覧を表示します。

## <a name="syntax"></a>構文

```
filesystems
```

## <a name="remarks"></a>コメント

-   この操作を成功させるのには、ボリュームを選択してください。 使用して、 **ボリュームを選択して** コマンドのボリュームを選択し、それにフォーカスをします。
