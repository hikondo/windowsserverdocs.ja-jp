# <a name="offline"></a>なっ



オンラインのディスクまたはボリュームをオフライン状態にかかります。

## <a name="syntax"></a>構文

```
Offline disk
Offline volume
```

## <a name="parameters"></a>パラメーター

|パラメーター|説明|
|---------|-----------|
|[オフラインディスク](offline-disk.md)|オフラインの状態にフォーカスがあるオンライン ディスクを移動します。|
|[オフラインボリューム](offline-volume.md)|オフラインの状態に、オンラインのボリュームがフォーカスを移動します。|
