# <a name="nslookup-set-debug"></a>nslookup set debug



デバッグモードをオンまたはオフにします。

## <a name="syntax"></a>構文

```
set [no]debug
```

## <a name="parameters"></a>パラメーター

|  パラメーター  |                         説明                          |
|-------------|--------------------------------------------------------------|
| **nodebug** | デバッグモードをオフにします。 既定の構文は**nodebug**です。 |
|  **debug**  |                   デバッグモードをオンにします。                   |
|  \*\*{help  |                            ?}\*\*                            |

## <a name="remarks"></a>注釈

-   デバッグモードを有効にすると、サーバーに送信されたパケットと結果の解答に関する詳細情報が出力されます。
