# <a name="vol"></a>vol



存在する場合は、ディスクのボリューム ラベルとシリアル番号を表示します。  パラメーターを指定せずに使用する場合 **vol** 、現在のドライブ情報が表示されます。

## <a name="syntax"></a>構文

```
vol [<Drive>:]
```

### <a name="parameters"></a>パラメーター

|パラメーター|説明|
|---------|-----------|
|\<Drive>:|ボリューム ラベルとシリアル番号を表示するディスクが含まれるドライブを指定します。|
|/?|コマンド プロンプトにヘルプを表示します。|
