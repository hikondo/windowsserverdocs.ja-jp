# <a name="bitsadmin-cache-and-getexpirationtime"></a>bitsadmin cache および getexpirationtime

キャッシュの有効期限を取得します。

## <a name="syntax"></a>構文

```
bitsadmin /cache /getexpirationtime
```

## <a name="examples"></a>例

キャッシュの有効期限を取得するには:

```
bitsadmin /cache /getexpirationtime
```
