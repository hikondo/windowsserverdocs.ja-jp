# <a name="bitsadmin-cache-and-getexpirationtime"></a>bitsadmin cache と getexpirationtime



キャッシュの有効期限を取得します。

## <a name="syntax"></a>構文

```
bitsadmin /Cache /GetExpirationTime 
```

## <a name="BKMK_examples"></a>例

次の例では、キャッシュの有効期限を取得します。
```
C:\>bitsadmin /Cache /GetExpirationTime
```
