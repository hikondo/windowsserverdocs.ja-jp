# <a name="begin-backup"></a>begin backup

> 適用対象: Windows Server 2012 R2、Windows Server 2012

完全バックアップセッションを開始します。 このコマンドは、既定のコピーバックアップ設定よりも優先されます。

## <a name="syntax"></a>Syntax

```
begin backup
```
