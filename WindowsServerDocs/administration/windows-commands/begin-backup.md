# <a name="begin-backup"></a>バックアップの開始



完全バックアップセッションを開始します。

## <a name="syntax"></a>構文

```
begin backup
```

## <a name="remarks"></a>コメント

-   このコマンドは、既定のコピーバックアップ設定よりも優先されます。
