# <a name="bitsadmin-util-and-help"></a>bitsadmin util とヘルプ



**Util**スイッチのコマンドラインの使用法を表示します。

## <a name="syntax"></a>構文

```
Bitsadmin /Util /Help 
```

## <a name="BKMK_examples"></a>例

次のコマンドは、 **Util**スイッチのコマンドラインヘルプを表示します。
```
Bitsadmin /Util /Help
```
