# <a name="nslookup-set-d2"></a>nslookup set d2

詳細デバッグモードをオンまたはオフにします。 すべてのパケットのすべてのフィールドが出力されます。

## <a name="syntax"></a>構文

```
set [no]d2
```

### <a name="parameters"></a>パラメーター

| パラメーター | Description |
| ---------- | ---------- |
| nod2 | 詳細デバッグモードをオフにします。 これが既定値です。 |
| d2 | 詳細デバッグモードをオンにします。 |
| /? | コマンド プロンプトにヘルプを表示します。 |
| /help | コマンド プロンプトにヘルプを表示します。 |
